Proyecto: Image Annotation Practice

Herramienta utilizada: CVAT

Número de imágenes: 50

Etiquetas utilizadas:
- person
- car
- animal

Tipo de anotación:
Bounding Boxes (Object Detection)

Descripción:
Se realizó la anotación manual de 50 imágenes que contienen personas,
animales y vehículos utilizando la herramienta CVAT.
Las cajas delimitadoras fueron creadas siguiendo buenas prácticas
de anotación para entrenamiento de modelos de visión por computadora.

Formato de exportación:
COCO dataset
